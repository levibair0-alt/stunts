## Getting Started

### Prerequisites
- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation
```bash
# Clone the repository
git clone https://github.com/levibair0-alt/stunts.git

# Navigate to project directory
cd stunts

# Install dependencies (when available)
npm install
```
